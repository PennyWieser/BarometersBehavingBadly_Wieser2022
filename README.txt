This repo contains the supporting information for the Journal of Petrology Paper:
"Barometers behaving badly I: Assessing the influence of analytical and experimental uncertainty on 
clinopyroxene thermobarometry calculations at crustal conditions", Wieser et al. 2023. 
Paper doi: https://doi.org/10.1093/petrology/egac126

If you want the ArcPL dataset, you can get an updated version at 
https://github.com/PennyWieser/BarometersBehavingBadly_PartII
As some experiments were published between part I and part II

The latest version of the code can be found archived on Zenodo

Please contact Penny Wieser with any questions, or raise an issue on GitHub!
